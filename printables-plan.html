Step 1 — Set Up Your Google Sheet Catalogue

This sheet is your "database". Each row = one printable on your website.

  1. Go to sheets.google.com → Create New Sheet
  2. Name it: GigNLearn Printables
  3. Create these exact columns in Row 1:
A: titleB: categoryC: emojiD: drive_idE: preview_idF: ageG: tags
ABC Tracing SheetAlphabets🔤 1abc...XYZ1img...ABC3-5abc,tracing,letters
Numbers 1-10Numbers🔢 1def...PQR1jkl...DEF3-6numbers,math,tracing
Vowels WorksheetAlphabets📝 1ghi...MNO1mno...GHI4-6vowels,abc,writing
The drive_id is the long code in your Google Drive file's share link. e.g. https://drive.google.com/file/d/1abc...XYZ/view
2

Step 2 — Make Drive Files Viewable

For each PDF/image in your Drive:

  1. Right-click the file → Share
  2. Click "Anyone with the link"
  3. Set permission to Viewer
  4. Click Copy Link — extract the ID from the URL
Do this for both the PDF file AND a thumbnail/preview image (PNG/JPG) for each worksheet.

Two useful Drive URL formats:

// Direct PDF download link: https://drive.google.com/uc?export=download&id=FILE_ID // Inline preview (embed in browser): https://drive.google.com/file/d/FILE_ID/preview // Thumbnail image: https://drive.google.com/thumbnail?id=FILE_ID&sz=w400
3

Step 3 — Publish Sheet as JSON API (FREE!)

Google Sheets has a built-in way to serve your data as JSON — no backend needed!

  1. In your Google Sheet → click File → Share → Publish to web
  2. Select Sheet 1 → Choose Comma-separated values (.csv)
  3. Click Publish
  4. Copy the URL — it looks like:
    https://docs.google.com/spreadsheets/d/SHEET_ID/pub?output=csv
Your printables.html will fetch this CSV every time someone visits, so adding a new row to your sheet INSTANTLY adds a new printable to your website! No code changes needed.

Step 4 — Your printables.html Code

This is the code for your printables page. Replace YOUR_SHEET_ID with your actual Google Sheet ID:

const SHEET_URL = 'https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/pub?output=csv'; // Fetch printables from Google Sheet async function loadPrintables() { const res = await fetch(SHEET_URL); const csv = await res.text(); const rows = parseCSV(csv); // skip header row rows.forEach(row => { const { title, category, emoji, drive_id, preview_id, age, tags } = row; createCard({ title, category, emoji, drive_id, preview_id, age, tags }); }); } // Create a printable card with preview + download function createCard(item) { const previewURL = `https://drive.google.com/thumbnail?id=${item.preview_id}&sz=w400`; const downloadURL = `https://drive.google.com/uc?export=download&id=${item.drive_id}`; const embedURL = `https://drive.google.com/file/d/${item.drive_id}/preview`; // Card HTML — preview image + modal + download button card.innerHTML = ` <img src="${previewURL}" alt="${item.title}"> <h3>${item.emoji} ${item.title}</h3> <button onclick="openPreview('${embedURL}')">👁 Preview</button> <a href="${downloadURL}" download>⬇ Download PDF</a> `; }

The page features:

  • 🔍 Search by title or tag
  • 📂 Filter by category (Alphabets, Numbers, Colors, Animals...)
  • 👁️ Preview worksheet in a popup modal before downloading
  • ⬇️ One-click download directly from Google Drive
  • 📱 Mobile friendly grid layout

Step 5 — Your Daily Workflow (Super Easy!)

Once set up, adding new printables takes under 2 minutes:

  1. 📁 Upload new PDF to Google Drive → make it public → copy the file ID
  2. 🖼️ Upload a thumbnail image for it → copy that file ID too
  3. 📊 Open your Google Sheet → add ONE new row with the details
  4. 🌐 Visit your website → new printable appears automatically! 🎉
You can batch-add 10 new printables in one session by just filling 10 rows in the sheet. The website updates instantly.
